# python-calculator
#Simple calculator using functions

This is a simple calculator project developed using Python functions.
It performs basic arithmetic operations like Addition, Subtraction, Multiplication, and Division.
This project is suitable for beginners and helps in understanding functions, user input, and conditional statements in Python.


Features
Uses functions for each operation
User-friendly menu-driven program
Handles division by zero
Beginner-friendly and clean code


Technologies Used
Language: Python
Concepts:
Functions
Conditional statements (if-elif)
User input
